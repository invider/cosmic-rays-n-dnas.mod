# cosmic-rays-n-dnas.mod
Global Game Jam 2020 Entry

# How to Run

Install collider.jam framework:

```
npm install -g https://github.com/invider/collider.jam.git
```

Open *.mod folder and run

```
jam
```

Run in develop mode with hot code reload

```
jam -d
```

Skip splash screens

```
jam -d --fast
```
